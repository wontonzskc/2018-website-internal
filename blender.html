For the head of the hammer, I started with a cylinder with a mirror multiplier so that I only need to model one side of the hammer and blender will automaticly generate the other side for me. I then used extract and scale to model the hammer. I also used the subdivision surfaces multiplier to make sure that the cylinder part of the model is actually a cylinder instead of a polygon. The reason why I didn't just give the cylinder more surfaces when I created it is because that will increase the amount of vertices that I have to work with, which will make everything slower and harder to model. The subdivision surfaces multiplier only increases the amount of surfaces when the model is being rendered. I set it to increase the surfaces by 5 times when rendered. I had to use loop cuts to restrict the multiplier so that it doesn't smooth out the model too much. I had to make two loop cuts around the edges that I wanted to be sharp, so that the multiplier didn't smooth them out completely. I used the same method to model the handle of the hammer.


I used the knife tool to cut out edges so that I can extrude the surfaces downwards to create the pattern on the ring between the head of the hemmer and the handle. I then used the simply deform and array multipliers to duplicate and bend the blocks of pattern so that it fits perfectly around the dent I made on the hammer head.


I created a small rectangle and used the spin tool to duplicate it. I then rotated the ring and resized it to fit in the hammer head. I duplicated the ring and resized, and stretched them to fit deeper into the hammer, creating the turbo. I used the same method to create the ring of spikes around the cone that's in the middle of the turbo.


For the textures, I used the node editor inside the cycles render engine that comes with blender. I created my own node group that used Physics based rendering (PBR) to simulate the fresnel, reflection, index of refraction and so on. I used a normal map that I downloaded off the internet to add dents and scratches to my model. I also created a node group with the exact RGBA code of metals that I thought I might use to make it easier for me to choose colors.


The glass case was made by creating a cube and stretching it to fit around the hammer and then creating a smaller rectangle on the base of the cube, extruding it upwards, creating a casing. I then used the glass BSDF node to create the material. I then made it appear as a diffused object to other meshes to minimize firefly creates in cycles render with a light path node.
